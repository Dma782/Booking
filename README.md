0. This is a personal project, a clone of booking.com, created to improve my skills in backend programming, Python, and FastAPI. It implements isolation levels, CRUD operations, and a bot using the FastAPI framework. The entire frontend is generated by AI and connected via CORS.


1. Create PostgreSQL database `database`.
2. Copy `.env.example` to `.env` and set database credentials.
3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Apply migrations:

```bash
alembic upgrade head
```

5. Run API:

```bash
uvicorn app.main:app --reload
```

6. Run Celery:
```bash
celery -A main_file worker --loglevel=info
```

- `POST /auth/register`
